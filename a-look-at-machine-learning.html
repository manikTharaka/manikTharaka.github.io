A look at machine learning

Date Mon 10 February 2014 Tags machine-learning / supervised learning /

Machine learning can be seen as a field of study that has roots in many fields such as computer science,statistics,mathematics,artificial intelligence and engineering.Practical application of machine learning can be experienced in your day to day life in instances such as spam detection in your e-mail account,voice recognition in smartphones and even in certain search engines, and the applications also span a wide range of fields such as physics,chemistry,biology,engineering and economics.

So what really is machine learning?Basically machine learning is concerened with the study of systems that can act without being explicitly programmed.In order to achieve this systems(computers) learn from data and modify or adapt their actions accordingly, once the learning or the training of the algorithm is achieved the system can take apropriate actions when presented with new data.The action in question can vary depending on the application of the algorithm from predicting a continuous numerical value or predicting the class label for a given set of data.The accuracy with which the system acts,or how close the to the actual value the learning algorithm can predict is known as the generalization ability of the algorithm.

To digest what we discussed so far let's look at a classic example, the problem of spam detection.Imagine your email account is equiped with a system to detect spam.In the beigning the system dosen't filter anything and you recieve spam mails as well as your regular correspondence.You have the ability to flag emails as spam,which you use to mark several spam mails.what happens now is that the system tries to learn what kind of features makes a spam email perhaps it will look at unusual combination of words,or for a certain set of words or maybe it will look at the sender of the email.Now that the system has learned or been trained what spam is,in the future it will filter out emails which it decides as spam.The accuracy with which the system detects spam is dependent on several factors such as the complexity of the learning algorithm used the number of examples or data used to train the algorithm.
